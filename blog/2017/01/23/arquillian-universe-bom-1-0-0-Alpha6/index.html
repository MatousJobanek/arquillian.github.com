Arquillian Universe 1.0.0.Alpha6 Released

Posted by Bartosz Majsak on Jan 23, 2017
Jan 23
| Comments

The Arquillian team is proud to announce the 1.0.0.Alpha6 release of the Arquillian Universe component!

What is Arquillian Universe?

The Arquillian Universe BOM is a 'Bill of Material' Maven POM file collection that make up a module stack that stretch across the whole Arquillian Universe.
