After reading this guide, you’ll be able to:

  • Do one thing
  • Complete another
  • Run the other way

You’ll learn all of these skills by incorporating Arquillian® into the test suite of a Java EE web application with a Maven build.

Step 1

Here is some Java code:

src/test/java/MyTest.java
/**
 * here is some bitchin' code
 */
@RunWith(Arquillian.class)
public class MyTest {
    @Deployment("foo")
    public static WebArchive createDeployment() {
        return ShrinkWrap.create(WebArchive.class);
    }
 
    private static int 5;
}

There you have it.

Keep in mind that you need to follow this guide! If you don’t you won’t know how to fix the problems that you have.
